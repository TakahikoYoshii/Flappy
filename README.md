### 説明
Flappy Birdという大人気アプリのクローンアプリです。
Firefox OS用のアプリとして作成しました、ブラウザでも動作します。

クローンアプリということで、少しキャラクターなどをアレンジしていますので、悪しからず。。
**※実際に登場するキャラクターは実在の人物に基づいています。**
